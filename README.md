# SnipAI

> 一款简洁高效的截图工具，开箱即用。

## 功能特性

### 📸 快速截图
- **全局快捷键** `⌘⇧A` 一键启动截图，随时随地响应
- **区域选择** 拖拽选取截图区域，8 个控制点精确调整
- **实时尺寸** 选区即时显示像素尺寸
- **RGB 取色** 鼠标移动实时显示颜色值

### ✏️ 标注工具
- **矩形** 绘制矩形框
- **椭圆** 绘制椭圆/圆形
- **箭头** 指向标注
- **画笔** 自由绘制
- **文字** 添加文本标注
- **马赛克** 遮蔽敏感信息
- **撤销/重做** 支持多步撤销

### 📌 贴图
- 将截图以悬浮窗口置顶显示
- 滚轮缩放、边缘拖拽调整大小
- 双击关闭、右键菜单操作

### 🎨 工具栏
- 深色胶囊风格设计，紫色主色调
- 工具切换、撤销、保存、确认一站式操作

### 💾 输出方式
- 自动复制到剪贴板
- 保存为 PNG / JPEG / BMP 文件

### ⚙️ 设置面板
- **通用设置** 开机启动、系统托盘、主题切换
- **快捷键** 自定义各项功能快捷键
- **外观** 颜色、线条粗细、圆角、毛玻璃效果
- **录屏** 系统声音、麦克风、视频格式与帧率
- **AI 设置** OCR 文字识别、智能抠图

## 开发中功能

### 🤖 AI 智能功能
> 🚧 正在开发中

AI 让截图从静态图片变成可分析、可识别、可翻译的智能内容 — 看到的不仅是画面，更是信息。

- **OCR 文字识别**
  截图中的文字直接识别提取。支持中英日韩多语言，离线引擎保护隐私。
- **AI 截图分析**
  把截图发给 AI，获得结构化分析 — UI 问题诊断、内容摘要、关键区域标注。
- **AI 多语言翻译**
  截图中的外语内容 AI 直接翻译，支持中英日韩等主流语言互译。
- **代码片段识别**
  截图里的代码 AI 识别并还原为可复制的代码文本，语法高亮，语言自动检测。

### 🔄 AI Agent 工作流
> 🚧 正在开发中

截图后在需要修改的地方逐个标注，一次性发送给 AI Agent — 它理解每一处标注的意图，统一处理并返回修改结果。不用再逐条描述、逐个修改。

**工作流程**：
1. 截图并标注多处
2. 一键发送给 AI Agent
3. AI 统一修改并返回

## 使用说明

1. 启动应用后，菜单栏出现托盘图标
2. 按 `⌘⇧A` 开始截图
3. 拖拽选择截图区域
4. 使用底部工具栏进行标注
5. 按 `Enter` 确认截图（自动复制到剪贴板并贴图）
6. 按 `Esc` 取消截图
