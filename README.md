# SnipAI

SnipAI 是一款面向 macOS 和 Windows 的截图、贴图、录屏与取色工具。它参考 Snipaste 的高频工作流，优先保证快捷键唤起、区域选择、复制、保存和贴图这些核心动作足够快、足够顺手。

## 已实现功能

### 截图

- 全局快捷键唤起截图。
- 支持区域截图、全屏截图和快速截图。
- 截图遮罩层覆盖全屏，选区内取消遮罩并保留原始画面。
- 选区可拖动、缩放和二次调整。
- 选区内双击可直接复制截图到剪贴板。
- 支持 `Enter` 复制、`S` 保存、`Esc` 取消。
- 截图后可复制到剪贴板、保存到本地，或直接创建贴图。
- 默认保存路径可在设置中修改。
- 默认图片保存目录：
  - macOS：`~/snipAi/Pictures`
  - Windows：`C:/snipAi/Pictures`

### 滚动截图

- 截图工具栏中提供滚动截图入口。
- 选择滚动截图后，用户通过鼠标滚轮手动延长截图区域。
- 截图区域触底后，遮罩层后方页面可继续滚动并拼接成长图。
- 支持 `Enter` 完成并复制长图，`Esc` 取消。

### 标注

- 支持画笔、空心矩形、箭头、文字标注。
- 支持马赛克涂抹，用于遮挡敏感信息。
- 支持颜色切换、撤销和完成标注。
- 标注内容会参与复制、保存和贴图输出。

### 贴图

- 支持把截图创建为置顶贴图窗口。
- 贴图可拖动、缩放和关闭。
- 鼠标滚轮可缩放贴图。
- 右键贴图可复制图像、图像另存为、移动到分组。
- 支持贴图分组管理。
- 托盘菜单中的贴图分组为单选状态，当前选中的分组会作为新贴图默认保存分组。
- 支持创建新贴图分组。
- 贴图管理窗口支持平铺展示已复制/保存到贴图库的截图。
- 贴图管理窗口支持按分组筛选，移动贴图到其他分组后会实时从当前分组视图中移除。
- 分组数量使用真实贴图数量统计。
- 支持一键贴出当前分组的全部贴图，并记住每个贴图上次移动后的位置和大小。
- 支持隐藏/显示全部贴图。
- 支持选中贴图后通过快捷键关闭全部贴图。

### 录屏

- 支持选择区域录屏。
- 选择区域后，需要点击悬浮工具栏中的录制按钮才开始录制。
- 录制中会显示醒目的红色录制边框和停止录屏控件。
- 录屏文件默认保存到可配置目录。
- 默认视频保存目录：
  - macOS：`~/snipAi/Movies`
  - Windows：`C:/snipAi/Movies`

### 取色器

- 支持从顶部工具栏进入取色器。
- 取色器以鼠标位置为放大镜视觉中心采样。
- 显示 HEX、RGB、HSL 颜色值和历史颜色。
- 鼠标左键选中颜色后自动复制 HEX 颜色值、退出截图模式，并显示提示。
- 支持为取色器配置全局快捷键。

### 设置与托盘

- 主窗口为设置窗口，关闭窗口后应用仍保留在系统托盘或 macOS 菜单栏。
- 托盘菜单支持开始截图、贴图分组、贴图管理、打开设置、退出应用。
- 设置窗口支持拖动。
- 快捷键设置支持修改截图、滚动截图、录屏、贴图、取色器等全局快捷键。
- 支持修改截图和录屏默认保存路径。
- 关于页展示应用版本与技术栈信息。

### 打包与发布

- 打包前会生成优化后的 `build/app`，压缩 HTML/CSS/JS 并使用 `asar` 封装应用代码。
- 打包产物会自动清理中间目录，只保留最终安装包。
- 当前默认输出：
  - macOS：`release/SnipAI-macOS-arm64.dmg`
  - Windows：`release/SnipAI-Windows-x64-Setup.exe`

## 默认快捷键

| 功能 | 快捷键 |
| --- | --- |
| 区域截图 | `F1` |
| 全屏截图 | `Control + Command + Shift + A` |
| 快速截图 | `Control + Command + D` |
| 滚动截图 | `Control + Command + S` |
| 区域录屏 | `Control + Command + R` |
| 取色器 | `Control + Command + C` |
| 快速贴图 | `F3` |
| 剪贴板贴图 | `Control + Shift + V` |
| 贴出当前分组全部贴图 | `Control + Command + Shift + S` |
| 隐藏/显示贴图 | `Control + Command + H` |
| 关闭全部贴图 | `Command + Esc` |

快捷键可以在设置页中调整。

## 权限说明

| 权限 | 用途 |
| --- | --- |
| 屏幕与系统音频录制 / 屏幕录制 | 截图、取色、录屏 |
| 辅助功能 | 全局快捷键、滚动截图时模拟滚轮 |

首次使用时，请在系统权限弹窗或「系统设置 -> 隐私与安全性」中允许 SnipAI。

## 本地开发

```bash
npm install
npm start
```

## 打包

```bash
npm run dist:all
```

打包完成后，`release/` 目录只保留 macOS 和 Windows 两个安装包。
