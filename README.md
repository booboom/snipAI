# SnipAI

> 一款简洁高效的截图工具，无需登录注册，开箱即用。

## 功能特性

### 📸 快速截图
- **全局快捷键** `⌘⇧A` 一键启动截图，随时随地响应
- **区域选择** 拖拽选取截图区域，8 个控制点精确调整
- **实时尺寸** 选区即时显示像素尺寸
- **RGB 取色** 鼠标移动实时显示颜色值

### ✏️ 标注工具
- **矩形** 绘制矩形框
- **椭圆** 绘制椭圆/圆形
- **箭头** 指向标注
- **画笔** 自由绘制
- **文字** 添加文本标注
- **马赛克** 遮蔽敏感信息
- **撤销/重做** 支持多步撤销

### 📌 贴图
- 将截图以悬浮窗口置顶显示
- 滚轮缩放、边缘拖拽调整大小
- 双击关闭、右键菜单操作

### 🎨 工具栏
- 深色胶囊风格设计，紫色主色调
- 工具切换、撤销、保存、确认一站式操作

### 💾 输出方式
- 自动复制到剪贴板
- 保存为 PNG / JPEG / BMP 文件

### ⚙️ 设置面板
- **通用设置** 开机启动、系统托盘、主题切换
- **快捷键** 自定义各项功能快捷键
- **外观** 颜色、线条粗细、圆角、毛玻璃效果
- **录屏** 系统声音、麦克风、视频格式与帧率
- **AI 设置** OCR 文字识别、智能抠图

### 🚫 无订阅
- 无登录注册，无订阅机制
- 完全本地运行，无需网络

## 使用说明

1. 启动应用后，菜单栏出现托盘图标
2. 按 `⌘⇧A` 开始截图
3. 拖拽选择截图区域
4. 使用底部工具栏进行标注
5. 按 `Enter` 确认截图（自动复制到剪贴板并贴图）
6. 按 `Esc` 取消截图

## 技术栈

- Python 3.9 + PyQt6
- macOS Quartz / AppKit（全局热键 & 屏幕捕获）
- PyInstaller（应用打包）
